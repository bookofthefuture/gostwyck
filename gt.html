The GT Project

Acquired in 2023, the donor BMW Z3 1.9 automatic represents the foundation for our most ambitious electric conversion yet. This pristine example provides the perfect base for creating a sophisticated grand tourer that can handle both daily driving and extended continental journeys.

The GT project will incorporate all the lessons learned from the Spyder build, with enhanced performance, range, and comfort. The hardtop configuration makes it suitable for year-round use, while the planned fast-charging capability opens up new possibilities for electric touring.

Development Timeline

  • 2023: Donor vehicle acquired
  • 2024: Planning and component research
  • 2025: Build commencement (Winter)
  • 2026: Target completion
